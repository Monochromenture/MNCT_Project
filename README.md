# MONOCHROMETURE 
#2D橫向平台 #冒險 #遊戲


## 專題介紹

黑白線條的主角們闖入原本充滿色彩的世界，尋找回去原本世界的方法。</br>
劇情偏荒誕搞笑無厘頭的風格。

## 使用工具 (程式部分)
- Unity
- Visual Studio

## 實作功能
#### 玩家操控
- 移動控制 (左右移動、跳躍)
- 鏡頭跟隨玩家移動
- 手持武器攻擊

#### UI
- 主選單、暫停選單製作
- 血量狀態顯示

#### 關卡
- 跳躍平台
- 移動平台 (固定範圍移動)
- 尖刺陷阱
- 敵人 (固定範圍移動)
- Checkpoint及重生位置
- 關卡傳送門

#### 互動
- 玩家攻擊敵人
- 敵人碰撞攻擊

#### 動畫/音效
- 置入開頭/結尾動畫
- 進入傳送門，玩家原地旋轉消失
- 獲得關鍵道具，發光飛走動畫
